# 풀림 스터디 14→8 기능 통합 (Feature Consolidation)

## 목표
14개 기능을 학생 인지 부하·정보 밀도·핵심 가치 중복 기준으로 8개로 통합. 라우트 변경 + 학생 UI 명칭 갱신 + redirect 안전망 구축.

## 비고

### 통합 결정 근거
- **인덱스 + 풀이분석**: 둘 다 "내 학습 상태 분석"이 핵심 — 탭 2개로 정보 분리만으로 충분
- **오답정복 + 기억장치**: 둘 다 복습 영역 — Leitner 5-box(정복) + 망각곡선(전체)으로 분담
- **튜터 + 코치**: 둘 다 AI 대화 — 힌트 모드(튜터) + 에이전트 호출(코치) 탭으로 분리
- **모의고사 → 무한풀기**: 시험 모드 토글이면 충분, 별도 surface 불필요
- **수학랩 → 비주얼**: VLM의 handwriting 어댑터로 흡수 가능
