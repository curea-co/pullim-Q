# 공스타그램 일간 시간표 레퍼런스 50장 분석

## 개요

풀림 플래너의 현재 일간 시간표가 시계 모양인데, 이 형태를 대체할 디자인 방향을 잡기 위해 공스타그램 레퍼런스 50장을 모아 분석한 보고서다. plan 작성과 개발은 하지 않았고, 정보 수집과 패턴 추출만 수행했다.

검색은 Pinterest에서 두 쿼리로 했다.
- `공스타그램 시간표`
- `공스타그램 플래너`

수집은 2026-05-08, 한국어 Pinterest(`kr.pinterest.com`) 기준이다. 이미지 50장은 [`images/`](images/)에, 출처 URL은 [`urls.txt`](urls.txt)에 보존했다.

## 한눈에 보기

- **시계형은 비주류였다.** 50장 중 시계형 일간 플래너는 1장(33번)뿐. 사용자의 거부 방향이 트렌드와 일치한다.
- **일간 시간표의 표준은 "좌 과목별 to-do + 우 24시간 사이드 트래커"** 다. 11장이 이 구성을 거의 그대로 따른다 (04, 11, 20, 23, 24, 25, 32, 35, 43, 44, 그리고 38번 카드의 Focus 자료).
- **시간 시각화의 핵심은 형광펜 색칠**. 시간 셀에 가로 막대로 채워 "이만큼 했다"를 드러내는 방식이 거의 전부다.
- **컬러는 두 축으로 갈린다.** 꾸미기 축(라벤더·세이지·라이트블루·파스텔·캐릭터)과 미니멀 축(흑백·다크). 한 앱에서 양쪽을 다 잡으려면 테마 스킨이 자연스럽다.
- **인스타 공유 카드형이 새 결**. 38, 41–48번처럼 4분할(Wake-up / Meals / Exercise / Focus) 카드로 일과를 회고하는 포맷이 등장. 풀림이 "공스타그램 → 풀림" 브랜딩을 직접 가져갈 수 있는 후보.

## 1. 형식 분포

| 형식 | 개수 | 대표 번호 |
|---|---|---|
| 주간 매트릭스 (요일 × 교시) | 18 | 02, 05, 06, 08, 09, 12, 13, 14, 15, 18, 19 |
| 일간 페이퍼 (좌 to-do + 우 24h 사이드) | 11 | 04, 11, 20, 23, 24, 25, 32, 35, 43, 44 |
| 인스타 공유 카드형 (4분할) | 8 | 38, 39–42, 45–48 |
| 학습 노트 / 요약 | 5 | 26, 27, 36, 49 |
| 30일 챌린지 / 트래커 | 4 | 16, 22, 37 |
| 월간 캘린더 | 2 | 17, 29 |
| 만다라트 9×9 | 2 | 21, 31 |
| 24h 컬러블록 (루틴 시각화) | 1 | 28 |

→ **시계형은 50장 중 1장**. 일간 페이지는 사이드 24h 트래커가 사실상 표준이고, 주간 페이지는 매트릭스가 표준이다.

## 2. 일간 시간표 표준 IA

가장 빈번하게 등장하는 일간 페이지 구성(04, 11, 20, 23, 24, 25, 32, 35, 43, 44):

```
┌───────────────────────────────────────────────────┐
│  Date · D-day · Total time · Comment              │  ← 헤더
├──────────────────────┬────────────────────────────┤
│                      │                            │
│   과목별 To-do       │   24h Time Table           │
│   (Subject ·         │   (1시간 6칸 = 10분 단위    │
│    Detail · ✓)       │    또는 4칸 = 15분 단위)    │
│                      │                            │
│                      │   → 막대 색칠로 시간 시각화 │
│                      │                            │
├──────────────────────┴────────────────────────────┤
│  Memo / Notes                                     │
└───────────────────────────────────────────────────┘
```

핵심 단위:
- **24h 사이드 그리드의 시간 범위**는 두 갈래다. (a) 06시~다음날 01시 (학생 일과 중심, 04·11·23번), (b) 00~24시 풀 (20·44번).
- **분 단위 셀**은 보통 1시간 6칸(10분) 또는 4칸(15분).
- **색칠**은 형광펜 막대형. 한 시간을 가로 막대로 채워 "이 시간대에 이 과목 했다"를 표현. 24·32·35번에서 라벤더/핑크 색칠이 시그니처.
- **헤더 위젯**: D-day (수능/시험), Total time (오늘 누적), Comment (영문 한 줄), Day n/66 (챌린지 진행도). 거의 모든 일간 페이지가 이 조합 안에서 골라 쓴다.

## 3. 주간 시간표 표준 IA

요일 × 교시 매트릭스가 압도적이다(18장). 시계형 주간 시간표는 0건.

세부 변형:
- **헤더 라벨**: `MON TUE WED THU FRI` (영문) 또는 `월 화 수 목 금` (한글). 영문이 미세하게 더 많다.
- **세로축**: 1교시~7교시 또는 시간(06:00~12:00).
- **셀 정보**: 과목명(필수) + 교수/선생 이름 + 강의실/교실 코드 (옵션). 13·05·09번이 풀 정보 패턴.
- **컬러**: 셀 배경색 또는 좌측 컬러 도트 라벨로 과목을 구분. 대학생 풍은 주로 셀 배경색(09·14), 중고등학생 풍은 컬러 도트(13).

## 4. 시각 언어 — 컬러 팔레트 클러스터

| 클러스터 | 분위기 | 대표 |
|---|---|---|
| 라벤더 / 페일 퍼플 + 화이트 | 정갈, 여학생 인기 | 16, 22, 24, 25, 32 |
| 세이지 그린 + 베이지 | 차분, 식물성 | 03, 19, 33, 38 |
| 라이트 블루 + 그레이 | 깨끗, 소년 인기 | 08, 12 |
| 흑백 미니멀 | 노션·Apple 풍 | 02, 11, 18, 20, 29 |
| 다크 + 화이트 텍스트 | 노트앱·집중 | 05, 14, 27, 36 |
| 핑크 형광 + 화이트 | 부드러움, 필기 | 32, 35 |
| 다중 파스텔 (셀별 색) | 캐릭터, 초딩 풍 | 06, 09, 13, 17 |
| 형광펜 컬러칠 (노트 위) | 손글씨 핵심 | 21, 23, 24, 32, 44 |

→ "라벤더 + 사이드 24h 트래커"가 공스타그램 시그니처에 가장 가깝다.

## 5. 타이포 패턴

- **헤더**: Bold 산세리프 (`MY MONTHLY PLAN`, `Time table`, `TODAY`) 또는 손글씨 (`Saturday`, `Tuesday`).
- **영문 라벨이 압도적**. `Mon Tue Wed Thu Fri`, `Time table`, `To-do List`, `Memo`, `D-day`, `Total time`, `Comment`. 한글은 키워드(시간표·플래너·계획)만 큼직한 폰트로 강조.
- **셀 안 폰트**: 작은 산세리프 또는 직접 쓴 손글씨. 둘 다 흔하다.
- **영문 슬로건 한 줄**: `Silent move loud results` (24번), `Remember that moment, turn on like a light` (10번) 같이 한 줄 동기 부여 멘트. 거의 모든 일간 페이지에 코멘트 칸이 있다.

## 6. 데코 / 캐릭터 — 두 축

- **꾸미기 축**: 산리오(쿠로미: 04, 25), 동물 일러스트(고양이·곰·햄스터·토끼·판다: 12, 13, 08, 10), 손그림 스티커(꽃·별·구름·체리: 06, 32), 인터넷 밈 폭발 말풍선(01).
- **미니멀 축**: 장식 0, 그리드 라인만, 노션·Apple Calendar 풍 (02, 11, 18, 20, 29).

→ 한 앱에서 둘 다 만족시키려면 테마/스킨 시스템이 자연스럽다. 풀림이 어느 쪽으로 정체성을 잡을지가 기획 결정 포인트.

## 7. 시간 시각화 관용구

| 종이 위 관용구 | 디지털 번역 후보 |
|---|---|
| 형광펜으로 셀 색칠 → 공부 시간 표시 | 셀 탭/드래그로 시간 슬롯 채우기 (Toggl·Forest 류) |
| 과목별 컬러 코드 | 과목 단위 컬러 토큰 + 셀 자동 채색 |
| Total Time / D-day / Day n/66 | 헤더 위젯 — 누적 시간, 디데이, 챌린지 진도 |
| 영문 슬로건 코멘트 | "오늘의 한 줄" 입력 칸 (선택) |
| 30일 / 66일 챌린지 격자 | 해빗 트래커 모듈 (별·원·체크) |
| 만다라트 9×9 | 큰 목표 → 8개 하위 목표 → 64개 액션 (장기 목표 페이지) |
| 인스타 4분할 카드 (Wake-up/Meals/Exercise/Focus) | 일과 회고 공유 카드 — 공유·저장 기능과 결합 |

## 8. 셀 정보 구조 — 풀림 데이터 모델 매핑

주간 시간표 셀에 들어가는 정보 (13·09·14):
- 과목 (필수)
- 교수/선생 이름
- 강의실/교실 코드
- 컬러 라벨 (과목 분류)

일간 to-do 행 (24·25·32):
- 과목 분류 (좌측 라벨, 예: 수학·영어·국어)
- 세부 task (교재명, 범위, 강의 회차)
- 체크 표시 (○ / △ / × / V)
- 누적 시간 또는 D-마크

→ 풀림 플래너의 블록 구조 (`linkedFeatureSlug` / `subject` / `detail` / `status`) 와 1:1 매핑 가능하다. 데이터 모델 큰 변경 없이 시각만 바꾸는 길이 열려 있다.

## 9. 풀림 일간 시간표 재설계 시사점 6가지

(plan 단계 아님 — 사용자 결정용 옵션 카드)

1. **시계형 → 사이드 24h 그리드 전환**. 50장 중 압도적 표준. 좌측 과목별 to-do + 우측 24시간(또는 06–01시) 분 단위 셀.
2. **셀 색칠 인터랙션**. 형광펜의 디지털 번역. 빈 셀을 탭/드래그하면 진행 중 과목 컬러로 채워지고 → 누적 시간 자동 계산.
3. **과목 컬러 토큰**. 풀림 도메인/과목별 색을 토큰으로 정의해 자동 채색. 사용자는 색을 직접 안 골라도 됨.
4. **헤더 위젯 셋트**. D-day(수능·시험), Total time, Day n/66 챌린지 진행도, 한 줄 코멘트. 모두 옵셔널 토글로.
5. **테마 시스템**. 라벤더 / 세이지 / 라이트블루 / 흑백 / 다크 5종 정도. 꾸미기 축과 미니멀 축을 한 입력으로 커버.
6. **공유 카드 export (선택)**. 38번처럼 일과 끝에 4분할 카드(공부·식사·운동·회고)를 SNS 공유 이미지로 export. 공스타그램 정체성과 직결되며 풀림 브랜딩 후보.

## 10. 후속 수집 권장

이번 라운드에는 안 했지만 다음 단계에서 보면 좋은 항목:

- **"스터디플래너 앱" 검색** — 디지털 구현 사례 (열품타·트레인·노션 템플릿·Notion 학생용 다이어리). 공스타그램의 종이 형식이 모바일에서 어떻게 변형되는지 확인.
- **"스터디 다이어리 손그림" 검색** — 손글씨/스티커 스타일 미세 디테일.
- **원형 도넛 차트 (라이프 시각화)** — 시계형은 거부됐지만 도넛 차트는 풀림 분석 페이지에 응용 여지가 있다. 별도 수집 권장.
- **인스타그램 본진** — 로그인 후 직접 검색하면 카드형 트렌드(38번 결)가 더 풍부할 가능성.
