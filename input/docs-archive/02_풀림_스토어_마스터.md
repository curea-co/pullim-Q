# 풀림 스토어(Pullim Store) 마스터 문서

> **버전**: v1.0  
> **최종 업데이트**: 2026-04-09  
> **역할**: 풀림 3축 중 **유통(Distribution)** 축  
> **통합 출처**: 통합 기능 정의서, 핵심기능 정의서, 종합 프로덕트·기술 전략, 유통콘텐츠 전략

---

## 1. 전략적 포지셔닝

> **"검증된 교육 콘텐츠만 유통되는 프리미엄 마켓플레이스"**

스토어는 스튜디오에서 생산된 콘텐츠가 학습자에게 도달하는 **유통 채널**이자, 크리에이터에게 수익을 제공하여 생태계를 유지하는 **경제 엔진**이다. 아마존·쿠팡 같은 범용 마켓플레이스가 아닌, "풀림 인증"을 통과한 고품질 교육 콘텐츠만 유통되는 프리미엄 포지셔닝이 핵심이다.

## 3. 기능 상세

### 3.1 M1. 마켓플레이스 핵심

#### 주요 기능
- **전체 홈 필터**: 과목, 학년, 유형, 난이도, 가격대, 평점별 필터링
- **과목별 특화 홈**: 영어/국어/수학 등 과목별 전용 큐레이션 페이지
- **상품 상세페이지**: 미리보기(샘플 문항), 상품 설명, 리뷰, 크리에이터 정보, 관련 상품
- **결제**: 토스페이먼츠/아임포트 (신용카드, 카카오페이, 네이버페이)
- **장바구니·찜하기**: 관심 상품 저장 및 묶음 결제
- **주문 내역·배송 조회**: 디지털 상품 즉시 다운로드 + 실물 교재 배송 추적
- **팔로우**: 크리에이터 팔로우 → 신규 콘텐츠 알림

#### 기술 스택

| 구성 요소 | 기술 |
|----------|------|
| 프론트엔드 | Next.js + React (SSR/SSG로 SEO 최적화) |
| 결제 | 토스페이먼츠 / 아임포트 |
| 검색 | Elasticsearch + 시맨틱 검색 |
| 추천 | Collaborative Filtering + Content-Based |

---

### 3.2 M2. AI 큐레이션 검색

#### 교육과정 분류 체계 기반 시맨틱 검색
- "관계대명사 that vs. which 구분 문제"처럼 정밀한 교육 맥락 검색 가능
- 벡터 임베딩(OpenAI text-embedding-3-large / BGE-M3) + Pinecone + 교육과정 메타데이터 필터링
- 개념 검색 시 개념 이해 자료를 동시 제공

#### 경쟁사 비교

| 경쟁사 | 풀림 대비 약점 |
|--------|--------------|
| 크몽/탈잉 | 키워드 검색만 제공, 교육 맥락 이해 불가 |
| 비상 T샵/천재 T샵 | 마켓플레이스 형태 아님, 크리에이터 생태계 없음 |
| Teachers Pay Teachers | AI 없음, 한국 시장 미진출 |

---

### 3.3 M3. 크리에이터 스토어·판매관리

- **크리에이터 대시보드**: 상품별/자료별 판매 현황, 주문 내역, 정산 내역 시각화
- **정산 시스템**: 월간 정산, 판매 수수료 자동 공제 후 입금
- **프로필 관리**: 크리에이터 소개, 포트폴리오, 전문 과목/학년 표시
- **프로모션 도구**: 할인 쿠폰 발행, 한정 판매, 묶음 할인 설정

---

### 3.4 M4. 판매 스냅샷·저작권 보호

- **판매 시점 스냅샷**: 판매 시점의 콘텐츠를 불변(immutable) 스냅샷으로 보존. 원본 수정이 구매자에게 미반영.
- **워터마크·DRM**: 불법 복제 방지를 위한 워터마크 및 DRM 적용
- **문항 ID 참조 아키텍처**: 문항은 복사가 아닌 참조로 관리되되, 판매 시 스냅샷 분리

---

### 3.5 M5. 콘텐츠 품질 인증 시스템

- **풀림 인증 마크**: 스튜디오 검증 시스템(Self-Refine + 전문가 평가)을 통과한 콘텐츠에 품질 보증 마크 부여
- **AI 자동 품질 스코어링**: 콘텐츠 등록 시 자동 품질 점수 산출
- **사용자 리뷰**: 구매 후 리뷰 작성 가능
- **품질 모니터링**: 사후 품질 모니터링. 오류 신고 시 크리에이터에게 알림. 품질 점수 하락 콘텐츠 자동 비노출 처리.

---

### 3.6 M6. 구독형 콘텐츠 서비스

#### 구독 상품 유형

| 구독 상품 | 설명 | 가격 |
|----------|------|------|
| 매주 모의고사 | AI가 매주 신규 모의고사 자동 생성·배달 | ₩19,900/월 |
| Daily Drops | 매일 5~20문제 + 개념 + 단어 맞춤 배달 | ₩4,900/월 |
| 전 콘텐츠 올패스 | 스토어 전체 콘텐츠 무제한 접근 | ₩39,900/월 |

**핵심 가치**: AI 문제 생성으로 무한한 신규 콘텐츠 공급이 가능하여, 기존 문제집(유한 재고)과 차별화되는 "무한 구독" 모델이 가능.

---

### 3.7 M7. 학습 효과 데이터 기반 리뷰

> **세계 어디에도 없는 기능**

스터디에서 수집된 학습 데이터를 기반으로, "이 교재를 사용한 학생의 해당 유형 정답률 +15%" 같은 **실증 효과 데이터**를 콘텐츠에 자동 부착.

- 기존 마켓플레이스: 주관적 리뷰만 존재
- 풀림: 학습 데이터로 콘텐츠 효과를 **증명**

#### 구현 기술
- 스터디 학습 데이터 → A/B 분석 파이프라인 (교재 사용 전/후 성적 비교)
- 효과 뱃지 자동 생성

**활성화 시점**: 충분한 학습 데이터가 쌓인 이후 (추정 6개월 후)

---

### 3.8 M11. AI 추천 엔진 (개인화)

학습 진단 결과, 풀이 이력, 구매 이력, 유사 학습자 데이터를 종합하여 "지금 당신에게 가장 필요한 교재"를 개인화 추천.

**기술**: Collaborative Filtering + Knowledge Graph + 교육과정 분류 기반 Content-Based Filtering 하이브리드

**연구 근거**: Duolingo의 AI 개인화 추천이 일일 활성사용자를 51% 증가시킨 사례(2025).

## 8. 스토어 ↔ 3축 연동

### 8.1 ← 스튜디오 연동
- 스튜디오에서 제작한 콘텐츠를 즉시 스토어에 등록
- 검증 등급에 따라 "풀림 인증 마크" 자동 부여
- 판매 스냅샷 자동 생성

### 8.2 → 스터디 연동
- 구매한 교재를 스터디의 "내 서재"에서 즉시 온라인 학습 모드로 전환
- 스터디 진단 결과 기반 스토어 교재 추천 ("당신의 영어 어법 점수를 올려줄 교재 TOP 3")
- 교재 학습 후 해당 유형의 성적 변화를 측정하여 교재 효과를 수치화

### 8.3 ← 스터디 데이터
- 학습 데이터가 스토어의 "학습 효과 리뷰"에 활용
- 수요 기반 콘텐츠 생성 신호: 특정 유형의 문제가 부족하거나, 많은 학생이 특정 개념에서 어려움을 겪으면 스튜디오/크리에이터에게 "이 유형의 콘텐츠 수요가 높습니다" 알림

## 9. 시장 침투 전략

### Phase 1 — 앵커 콘텐츠 확보
- 풀림 시그니처 시리즈(자체 제작)로 스토어 기본 콘텐츠 확보
- 스타 크리에이터 10명 선별 영입 (독점 계약 + 수수료 우대)
- 무료 콘텐츠(진단 테스트, 샘플 문제 세트)로 학생 유입

### Phase 2 — 네트워크 효과 점화
- 크리에이터 수수료 차등 정책: 검증 등급 높을수록 낮은 수수료
- "매주 신 모의고사" 구독 상품 출시
- 학원 B2B 패키지(수업 키트 + 모의고사 묶음) 판매

### Phase 3 — 시장 표준화
- 교육 콘텐츠 유통의 표준 플랫폼 포지셔닝
