## 1. 서비스 개요

### 1.1 한 줄 정의
**플래너는 학생이 "내 공부를 내가 설계한다"는 자기주도 감각을 기르도록, 교육학 기반 엔진이 개인화된 학습 블록을 제안·실행·회고하게 해주는 학습 동반자다.**

### 1.2 포지셔닝
| 축 | Planner의 위치 |
|---|---|
| 주도권 | 학생 중심 (AI·부모·교사는 조력자) |
| 시점 | 학교 밖 시간 (방과 후·주말·시험 전) |
| 주 사용처 | 모바일 앱·태블릿 (홈 화면 위젯까지) |
| 경쟁 차별점 | 시간표 앱이 아니라 "교육학 엔진 + 감정 지능"을 담은 자기조절 도구 |

### 1.3 타겟 유저
1. **P1 (주) 학생**: 중2-고3, 자기주도 학습에 도전하는 학생
2. **P2 학부모**: 자녀 학습·감정을 존중하며 함께하고 싶은 부모
3. **P3 과외·학원 선생님**: 학생의 플래너를 공동 관리, 숙제 자동 배정
4. **P4 상담 선생님** (학교): 학생 웰빙 모니터링

### 1.4 핵심 가치 제안 (Value Props)
- **V1 · 교육학 기반**: 7대 학습과학 엔진(Spaced Repetition, Interleaving, Cognitive Load, Pomodoro, Active Recall, Deliberate Practice, Zeigarnik)을 블록 단위로 자동 적용
- **V2 · 감정 지능**: 번아웃 지표·컨디션 슬라이더로 계획을 컨디션에 맞춰 자동 조정
- **V3 · 집중 모드**: 폰을 덮지 않아도 되는 1:1 페어 프로그래밍급 집중 환경
- **V4 · 아름다운 공유**: 플래너 자체가 SNS 공유 가능한 카드·포스터로 즉시 변환
- **V5 · 함께 성장**: 부모·튜터·친구와 플래너를 공동 편집·응원

## 2. 디자인 원칙

1. **학생이 계획의 주인이다.** AI·부모·교사의 제안은 항상 학생의 확정 전 초안이다.
2. **공부 시간보다 공부 질을 먼저 본다.** 시간 채우기 UX는 금지, 블록 완료·정답률·감정이 핵심 지표.
3. **번아웃은 실패가 아니다.** "오늘은 쉴래요" 버튼은 1-click 가능, 이월된 블록은 자동 재배치.
4. **아름다움은 기본값이다.** 모든 화면은 스크린샷 공유에도 손색없어야 한다 (학생의 SNS 문화 존중).
5. **부모 접근은 초대제다.** 학생이 승인하지 않으면 부모는 어떤 데이터도 볼 수 없다.

## 4. 주요 기능 · 상세

### 4.1 플래너 빌더 (8단계 위자드)

학생이 자신의 학습 목표를 블록으로 변환하는 단일 진입 경로.

| # | 단계 | 입력 | 산출물 |
|---|---|---|---|
| 1 | 목표 설정 | 수능·내신·자격증 + D-day | 목표 프로필 |
| 2 | 과목·단원 | 과목 선택 + 약점 진단 | 학습 범위 |
| 3 | 가용 시간 | 요일별 시간 슬롯 + 고정 블록 | 시간 예산 |
| 4 | 학습 스타일 | MBTI-like 진단 (6문항) | 개인화 힌트 |
| 5 | 교육학 엔진 | 7개 엔진 중 ON/OFF 선택 | 엔진 설정 |
| 6 | 컨디션·난이도 | 기본 난이도 + 컨디션 슬라이더 | 블록 난이도 |
| 7 | 공유 설정 | 부모·튜터·친구 초대 여부 | 권한 모델 |
| 8 | 테마·꾸미기 | 7개 테마 · BGM · 락스크린 | 시각 프로필 |

**핵심 UX 요구사항**:
- 8단계 전체 완료 시간: 목표 10-15분 (첫 사용)
- "친구 플래너 복제" 버튼으로 5분 빌드 가능
- 각 단계 "나중에 정할게요" 허용 (Skip → 기본값)

### 4.2 7대 교육학 엔진 (Pedagogy Engine)

학습과학 연구 기반 · 블록 수준에서 자동 적용.

| 엔진 | 원리 | 적용 예 |
|---|---|---|
| **Spaced Repetition** | 망각 곡선 기반 간격 복습 | 어제 오답 단어를 오늘·3일 후·7일 후 노출 |
| **Interleaving** | 과목·유형 섞기 | 수학·영어·국어를 40분 간격으로 교차 |
| **Cognitive Load** | 작업 기억 부담 조절 | 개념 40분 → 10분 쉬고 문제 |
| **Pomodoro** | 25+5 분할 | 집중 25분 + 회복 5분 |
| **Active Recall** | 시험 효과 | 읽기 대신 "말로 설명"·"문제 풀이" 우선 |
| **Deliberate Practice** | 약점 집중 훈련 | θ 낮은 유형에 시간 가중 |
| **Zeigarnik Effect** | 미완성의 기억 강화 | 의도적 "끝내지 않고 중단"으로 다음 블록 동기 유지 |

**구현**:
- 플래너에 7개 엔진 중 선택 적용 (기본 3개 ON)
- 블록 카드에 엔진 이름 태그로 노출 (교육적 투명성)
- 학생이 "왜 이 엔진?" 버튼으로 원리 설명 요청 가능

### 4.3 홈 (Home)

**화면 구성**:
- 인사말 + 오늘의 블록 수 + 첫 블록까지 남은 시간
- **NEXT BLOCK 히어로**: 현재 다음 블록 카드 (CTA: "지금 시작하기")
- 오늘의 시간표 (7개 블록 미니 타임라인, 완료·진행·대기 상태)
- 주간 그래프 (θ 추세, 학습 시간)

**핵심 인터랙션**:
- NEXT BLOCK "5분 늦추기" 버튼 1-tap
- "블록 변경" → 오늘의 다른 블록과 스왑
- 시간표 드래그 앤 드롭으로 순서 변경

### 4.4 캘린더 (일/주/월)

**일간 뷰 — 24시간 원형 시간표 (Signature UX)**:
- 시계 모양 SVG, 24개 슬라이스
- 각 슬라이스: 과목 색상 + 이모지 + 짧은 레이블
- 현재 시각: 빨간 바늘
- 중심 허브: "오늘 계획 5h 10m · 3/7 완료 · D-342"
- 좁은 화면에서는 호버 툴팁으로 제목 노출

**주간 뷰**:
- 7열 × 24시간 캘린더 그리드
- 블록 색상 구분: 개념/문제/취약/암기/셀프설명/휴식

**월간 뷰**:
- 전통 월력 + 일별 블록 수 히트맵
- 시험일 D-day 오버레이

### 4.5 블록 실행기 (Block Executor) — 핵심 기능

한 블록을 실제로 수행하는 화면. 플래너의 심장.

**화면 구성**:
- 상단: 블록 제목 + 남은 시간 + 타이머
- 좌측: 개념 카드·교안 (로딩)
- 중앙: 작업 영역 (문제 풀이·녹음·타이핑)
- 우측: AI 스터디 버디 채팅 (Scope Guard 적용)
- 하단: 일시정지·완료·"오늘은 쉴래요" 버튼

**Pomodoro 통합**: 25분 후 자동 "회복 5분" 카운트다운, BGM 전환

### 4.6 집중 모드 (Focus Mode)

**목적**: 폰·디지털 방해 최소화하되 학생이 앱 안에 머물게.

**기능**:
- 타이머 전체 화면 (잠금)
- Lo-fi BGM·화이트 노이즈·빗소리 (8종)
- 수신 알림 차단 (시스템 권한)
- 부모·튜터는 "집중 중" 상태만 확인 (내용 불가)
- 몰입 깨지면 "왜?" 이유 선택 (산만·피곤·스트레스·다른 일) → 감정 로그
- 완료 시 XP 부여 + 스트릭 연장

### 4.7 AI 스터디 버디

**성격**: 친구 톤, 반말 가능, 이름·이모지 선택 가능.

**기능**:
- 블록 실행 중 대화 상대
- 개념 질문·응원·감정 반응
- Scope Guard 기본 L3 (학생이 L5로 올릴 수 있으나 이력 기록)
- 과외쌤 ClassBot과는 별개 (버디는 감정 중심)

### 4.8 내보내기 · 공유 (Export & Share)

**목적**: 플래너를 SNS·친구·부모와 공유 가능한 아름다운 이미지로 변환.

**5가지 포맷**:
| ID | 포맷 | 크기 | 용도 |
|---|---|---|---|
| F-01 | 오늘의 플래너 카드 | 1080×1080 | 인스타 피드 |
| F-02 | 주간 리포트 카드뉴스 | 1080×1080 × 5 | 카드뉴스 |
| F-03 | D-day 포스터 | A4 | 책상 프린트 |
| F-04 | 완료 인증 뱃지 | 600×600 GIF | 스토리 스티커 |
| F-05 | 연간 결산 | 1080×1920 × 10 | 인스타 스토리 Wrapped |

**기능**:
- 라이브 프리뷰
- 7개 테마 (파스텔·네온·모노·빈티지 등)
- 개인정보 마스킹 옵션 (이름 → 이니셜)
- 플랫폼 공유 (인스타·카톡·X·밴드·디스코드)

### 4.9 챌린지·리워드

**시스템**:
- XP: 블록 완료당 · 완료율·정답률에 따른 보너스
- Streak: 연속 일수 (🔥 이모지, 숨돌리기 카드 2개/월)
- 뱃지: 50종 (수학 100문제·새벽 기상·주말 풀커버 등)
- 시즌 챌린지: 친구 경쟁 또는 자체 목표

**UI**: 아바타·포토카드·친구 랭킹

### 4.10 꾸미기 (Decorate)

- 7개 테마 팔레트
- 8종 폰트 세트 (Pretendard/Bai Jamjuree/손글씨/레트로 등)
- 스티커 200종 + 사용자 업로드
- 블록 아이콘 200종
- 배경 이미지·패턴 30종
- 락스크린 위젯 · 홈스크린 위젯

### 4.11 성장 리포트 (Growth Report)

**주기**: 주간·월간·학기

**구성**:
- 학습 시간 vs 목표
- θ 추세 (과목별)
- 오답 정복 현황 (Q와 연동)
- 번아웃 지수 + 감정 로그
- 부모·튜터 전송 가능 (학생 승인 후)

### 4.12 설정·연동

**주요 연동**:
- 구글 캘린더 (양방향)
- 카카오톡 (알림)
- 애플 건강 (수면 데이터)
- 풀림 Q (오답 자동 복습 블록 생성)
- 풀림 ClassBot (학원 숙제 자동 흡수)

## 5. 핵심 사용자 플로우

### Flow A · 신규 학생 첫 플래너 생성
```
1. 앱 설치 → 로그인 (카카오/구글/애플)
2. 웰컴 화면 → "지금 만들기"
3. 8단계 위자드 (평균 12분)
4. 빌드 완료 → 오늘의 첫 블록 자동 배정
5. 홈 화면 진입 → NEXT BLOCK 안내
6. 블록 시작 → 완료 → XP +10
7. 첫 스트릭 시작 🔥
```

### Flow B · 블록 실행 풀 사이클
```
1. 홈 NEXT BLOCK "지금 시작"
2. 블록 실행기 진입
3. 집중 모드 자동 ON (선택)
4. 25분 집중 + 5분 회복
5. 완료 → 정답률 입력 → 감정 체크인
6. AI 버디가 다음 블록 준비 안내
7. 완료 카드 자동 생성 → 공유 여부 선택
```

### Flow C · 번아웃 감지 → 회복
```
1. 시스템이 3일 연속 피로 감지
2. 홈 화면 "오늘은 쉴래요?" 제안
3. 학생 수락 → 오늘 블록 자동 이월
4. 휴식 블록으로 대체 (산책·음악)
5. 내일 블록 난이도 20% 하향
6. 부모 알림 (학생 승인 시)
```

### Flow D · 학원 숙제 자동 흡수
```
1. 학원 선생님이 ClassBot에서 숙제 배정
2. Planner가 숙제 데이터 수신
3. 학생 가용 시간에 자동 블록 생성
4. 마감 24시간 전 자동 리마인드
5. 학생 완료 → 선생님에게 완료 알림
6. Q에서 오답 발생 시 오답정복 블록 자동 생성
```

### Flow E · 부모가 자녀 플래너 공동 관리
```
1. 학생이 플래너 공유 초대 전송
2. 부모 앱 설치 → 초대 수락
3. 부모 대시보드 (요약만, 상세 불가)
4. 부모 응원 메시지 전송
5. 주간 리포트 자동 수신
6. 학생이 공유 해제 시 즉시 접근 차단
```

## 6. 유스케이스

### UC-1 · 고3 수험생의 수능 100일
- 페르소나: 김나윤 (고2, 현재 고3 전환 준비)
- 문제: 수능 D-342, 과목 5개, 압박감
- 해결: 7개 엔진 ON + 번아웃 감지 + Q 오답 자동 복습
- 성과: 주 평균 학습 시간 32→40시간 (컨디션 유지), 수학 θ +0.3

### UC-2 · 중3 내신 집중 학생
- 페르소나: 이서준 (중3, 특목고 진학)
- 문제: 학원·과외·독학 시간표 충돌
- 해결: 구글 캘린더 + 학원 ClassBot + Planner 3-way 동기화
- 성과: 시간 충돌 주 0건, 공부 시간 효율 +25%

### UC-3 · 재수생의 루틴 재건
- 페르소나: 박민지 (재수, 독학)
- 문제: 선생님·동료 부재 → 자기관리 붕괴
- 해결: AI 버디 일일 체크인 + 공유 플래너 (친구 2명) + 번아웃 경보
- 성과: 하루 학습 중단율 38%→12%, 6개월 완주율 100%

### UC-4 · 과외 튜터의 학생 관리
- 페르소나: 이민서 (과외 10명)
- 문제: 학생별 진도·숙제 수기 관리
- 해결: 공유 플래너 + 숙제 자동 배정 + 주간 리포트
- 성과: 관리 시간 50% 감소, 학부모 신뢰도 상승

### UC-5 · 학부모의 자녀 이해
- 페르소나: 박○○ (자녀 고1)
- 문제: 자녀와 공부 얘기만 하면 갈등
- 해결: 공유 플래너로 자녀의 컨디션·노력 가시화 + 응원 메시지
- 성과: 갈등 감소, 월 1회 학습 대화 정착

## 7. 핵심 개념·시스템

### 7.1 블록 (Block) — 핵심 단위

```
Block {
  block_id: UUID,
  planner_id: UUID,
  subject: string,
  unit: string,
  title: string,
  start_time: timestamp,
  duration_min: int,
  pedagogy_engines: [engine_id],
  difficulty: 0.0-2.4 (θ scale),
  status: "pending" | "active" | "done" | "skipped" | "rescheduled",
  completion: {
    actual_start: timestamp,
    actual_end: timestamp,
    accuracy: 0-100,
    emotion: 1-5,
    interruptions: int
  }
}
```

### 7.2 번아웃 지수 (Burnout Index)

5개 지표의 가중 평균 (0-100, 낮을수록 위험):
1. 수면 시간 (연동 시)
2. 연속 블록 완료율
3. 감정 체크인 추세
4. 휴식 블록 수용률
5. "쉴래요" 버튼 사용 빈도

### 7.3 컨디션 슬라이더

학생이 매일 아침 피곤/보통/쌩쌩 선택 → 당일 블록 난이도 자동 ±20%.

### 7.4 감정 체크인 (Emotion Check-in)

- 주기: 매 블록 완료 후 · 하루 종료 시
- 옵션: 😊 😐 😔 😤 🥱 (5종)
- 저장: 민감 정보 → 학생 외 접근 금지 (공유 플래너에서도 기본 비공개)

### 7.5 공유 플래너 (Collaborative Planner)

- 학생이 주인, 타인은 초대받은 역할로 참여
- 역할: 부모·튜터·친구 (각각 다른 권한)
- 48시간 자동 승인 규칙 (학부모만, 다른 역할은 즉시 수락 필요)

## 13. 리스크·고려사항

### 13.1 심리·윤리
- **번아웃·우울 탐지의 한계**: AI는 진단이 아닌 "관찰", 위기 신호는 전문가 연계 의무
- **공유 플래너의 프라이버시**: 학생의 "거절권" 보장, 부모가 일방적으로 강제 접근 불가
- **도박화 우려**: XP·Streak이 강박으로 변질되지 않도록 "건전 알림" 내재화

### 13.2 기술
- **오프라인 우선**: 블록 실행은 오프라인에서도 가능해야 함
- **시간대 이슈**: 한국 표준시 고정, 유학생 대비 타임존 설정
- **배터리 최적화**: 집중 모드는 저전력 모드 우선

### 13.3 비즈니스
- **무료 티어**: 플래너 1개 + 주 20 블록
- **Standard ₩7,900/월**: 무제한 플래너, 공유, 내보내기 F-01~F-03
- **Plus ₩12,900/월**: 꾸미기 풀, F-04~F-05, AI 버디 고급
- **가족 플랜 ₩18,900/월**: 2학생 + 2부모
